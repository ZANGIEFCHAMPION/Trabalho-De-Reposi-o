 Trabalho-De-Reposição
Trabalho De Reposição Do 3ºDesenvolvimento De Sistemas site da Escola
Projeto EEPD-BH

Este projeto foi desenvolvido como recomposição escolar utilizando PHP Puro, MySQL, Bootstrap e outras tecnologias modernas.

Tecnologias Utilizadas
- HTML5  
- CSS3  
- Bootstrap  
- PHP  
- MySQL (XAMPP)  
- Apache (XAMPP)  
- Visual Studio Code  
- Git  
- GitHub  
- React.js (projeto poderá integrar futuramente)
- Vue.js (projeto poderá integrar futuramente) 
- Spring Boot (projeto poderá integrar futuramente)

---

Estrutura do Projeto
- index.php → página inicial com carrossel de imagens aleatórias  
- cursos.php → cursos da escola  
- blog.php → blog institucional  
- contato.php → formulário de contato  
- login.php → login de usuários  
- cadastro.php → cadastro  
- professores.php → área do professor  
- alunos.php → área do aluno  
- conectar.php → conexão com MySQL  
- /css/estilo.css → estilos do site  
- /js/script.js → scripts personalizados  
- /img/ → imagens usadas no carrossel
